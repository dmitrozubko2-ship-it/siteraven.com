# 🐦 Адмін панель Raven Family

## ✨ Що нового?

Адмін панель тепер включає **трьох основних розділи для керування контентом:**

### 📰 Керування новинами
- ✅ Додавання нових новин
- ✅ Редагування існуючих новин
- ✅ Видалення старих новин
- 🎨 Красивий інтерфейс з іконками та датами

### 👥 Керування членами сім'ї
- ✅ Додавання нових членів сім'ї
- ✅ Редагування інформації про члена (нік, ранг, фото)
- ✅ Видалення членів
- 🎖️ Підтримка 5 рангів: Лідер, Адміністратор, Офіцер, Учасник, Новобранець

### 📜 Керування правилами
- ✅ Додавання правил у різні категорії
- ✅ Редагування правил
- ✅ Видалення правил
- 📚 5 категорій: Загальні, Заборонено, Покарання, Вимоги, Ранги

---

## 🚀 Швидкий старт

### 1. Увійти до адмін панелі
```
URL: /admin
```

### 2. Виберіть розділ
- 📰 Новини - для керування новинами
- 👥 Члени сім'ї - для керування членами
- 📜 Правила - для керування правилами

### 3. Додайте нові елементи
Натисніть "➕ Нове" і заповніть форму

### 4. Редагуйте або видаляйте
Використовуйте кнопки ✏️ та 🗑️
