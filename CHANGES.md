### 📚 Нові методи в Admin.razor

#### Методи для новин:
- `OpenAddNewsForm()` - відкриває форму для додавання новини
- `EditNews(NewsItem news)` - редагує новину
- `SaveNews()` - зберігає новину
- `CancelNewsEdit()` - скасовує редагування
- `DeleteNews(int id)` - видаляє новину

#### Методи для членів:
- `OpenAddMemberForm()` - відкриває форму для додавання члена
- `EditMember(FamilyMember member)` - редагує члена
- `SaveMember()` - зберігає члена
- `CancelMemberEdit()` - скасовує редагування
- `DeleteMember(int id)` - видаляє члена
- `GetRankIcon(string rank)` - отримує емодзі для рангу

#### Методи для правил:
- `OpenAddRuleForm()` - відкриває форму для додавання правила
- `EditRule(RuleItem rule)` - редагує правило
- `SaveRule()` - зберігає правило
- `CancelRuleEdit()` - скасовує редагування
- `DeleteRule(int id)` - видаляє правило

### 🏗️ Нові класи моделей

- `NewsItem` - клас для новин
- `FamilyMember` - клас для членів сім'ї
- `RuleItem` - клас для правил
