## 📱 Адмін панель - Новий розділ "Керування контентом"

### Переваги нової функціональності:

1. **📰 Управління новинами**
   - Додавання нових новин з іконками та датою
   - Редагування існуючих новин
   - Видалення старих новин
   - Вся інформація зберігається в пам'яті компонента

2. **👥 Управління членами сім'ї**
   - Додавання нових членів із зазначенням рангу
   - Редагування інформації про члена
   - Видалення членів
   - Підтримка 5 рангів: Лідер, Адміністратор, Офіцер, Учасник, Новобранець

3. **📜 Управління правилами**
   - Додавання правил у різні категорії
   - Редагування правил
   - Видалення правил
   - Категорії: Загальні, Заборонено, Покарання, Вимоги, Ранги

### Як користуватися:

1. **Перейти на адмін панель** `/admin`
2. **Виберіть потрібний розділ:**
   - 📰 "Новини" - для керування новинами
   - 👥 "Члени сім'ї" - для керування членами
   - 📜 "Правила" - для керування правилами
3. **Натисніть "➕ Нове"** щоб додати новий елемент
4. **Заповніть форму** і натисніть "💾 Зберегти"
5. **Для редагування** натисніть "✏️"
6. **Для видалення** натисніть "🗑️"

## 🎨 Кастомізація

### Змінити кольори
Змініть CSS змінні у `app.css`:
```css
:root {
    --primary-color: #2c2c54;      /* Змініть сюди */
    --secondary-color: #0f3460;    /* Або сюди */
    --accent-color: #c41e3a;       /* Або сюди */
    --light-color: #e8e8e8;        /* Або сюди */
}
```

### Змінити дані у компонентах
Заперечте демонстраційні дані у `OnInitialized()` методах.

### Вимкнути звуки
У `raven-effects.js` знайдіть функцію `initHoverSounds()` та закоментуйте.

### Вимкнути летючих воронів
У `raven-effects.js` закоментуйте `startFlyingRavens()`.

## 🐛 Можливі проблеми та рішення

### Проблема: Звуки не працюють
**Рішення:** Деякі браузери блокують звуки, поки користувач не взаємодіє зі сторінкою. Це нормально.

### Проблема: Анімації повільні
**Рішення:** Залежить від потужності комп'ютера. Можна зменшити тривалість анімацій у CSS.

### Проблема: Демонстраційні дані не оновлюються
**Рішення:** Оновіть дані у методі `OnInitialized()` кожного компонента.

### Проблема: Теми не зберігаються
**Рішення:** Переконайтеся, що localStorage дозволяється у браузері.
