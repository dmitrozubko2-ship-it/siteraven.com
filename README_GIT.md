# 🐦 RAVEN FAMILY - GIT SETUP ✅ ГОТОВО!

## ⚡ СТАРТ ЗА 3 КОМАНДИ

```bash
# 1. Інітіалізувати
git init

# 2. Додати файли
git add .

# 3. Зробити commit
git commit -m "🐦 Initial commit: Raven Family"
```

## 🌐 ЗАЛИТИ НА GITHUB (опціонально)

```bash
git remote add origin https://github.com/твій-юзер/raven-family.git
git branch -M main
git push -u origin main
```
