# 🐦 Сім'я Воронів - Інструкція для всіх користувачів

## 🚀 Як запустити сайт

### Крок 1: Встановити .NET SDK 10.0
- Завантажити з: https://dotnet.microsoft.com/download
- Встановити версію **.NET 10.0**

### Крок 2: Запустити сервер
```bash
cd "Raven Family"
dotnet run
```

### Крок 3: Дозволити в брандмауері (Windows)
Коли перший раз запустиш, Windows може запитати дозвіл:
- Натисни: **Дозволити доступ**

---

## 🌐 Як дуступитися з інших пристроїв

### На ОДНІЙ машині:
```
http://localhost:5000
```

### З ІНШИХ пристроїв у ЛОКАЛЬНІЙ МЕРЕЖІ:

#### 1️⃣ Знайти IP адресу сервера

**Windows:**
```bash
ipconfig
```
Знайти рядок з `IPv4 Address:` (наприклад: `192.168.1.100`)

**Linux/Mac:**
```bash
hostname -I
```

#### 2️⃣ Відкрити в браузері іншої машини:
```
http://192.168.1.100:5000
```
(замість `192.168.1.100` введи свою IP адресу)

## ❓ Часті проблеми

### Проблема: "Не можу підʹєднатися"
**Рішення:**
1. Переконайся, що сервер запущений (дивись консоль)
2. Перевір IP адресу командою `ipconfig`
3. Переконайся, що обидві машини в ОДНІЙ мережі Wi-Fi

### Проблема: "Порт 5000 уже використовується"
**Рішення:**
```bash
# Завершити процес
lsof -ti:5000 | xargs kill -9  # Linux/Mac
netstat -ano | findstr :5000   # Windows
```

### Проблема: "Брандмауер блокує"
**Рішення:**
- Windows: Дозволи доступ до 5000 порту в Firewall
- Linux: `sudo ufw allow 5000/tcp`
