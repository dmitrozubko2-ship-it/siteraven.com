## ⌨️ Гарячі клавіші та функції

### На будь-якій сторінці:
- `🐦` Курсор змінюється на емодзі ворона
- `🌙` Клік на кнопку теми в меню - перемикання на світлу тему
- `🎵` Тихий звук при наведенні на кнопки

### На сторінці членів (/members):
- `🔍` Введіть нік у пошуковій строці
- `📊` Виберіть ранг у випадаючому меню
- `🔘` Клік на "Профіль" для деталей

### На сторінці галереї (/gallery):
- `🏷️` Клік на фільтр для сортування
- `📸` Клік на фото для лайтбокса
- `❌` Клік на X для закриття лайтбокса

## 🎯 Важливі компоненти та їх параметри

### 1. AchievementsBlock
```razor
<AchievementsBlock 
    ParticipantsCount="42"    <!-- Кількість учасників -->
    VictoriesCount="128"      <!-- Кількість перемог -->
    YearFounded="2021"        <!-- Рік創 -->
    RatingScore="4.9" />      <!-- Рейтинг -->
```

### 2. ReviewsCarousel
```razor
<ReviewsCarousel />  <!-- Без параметрів, використовує внутрішні дані -->
```

### 3. NavLink (у меню)
```razor
<NavLink class="nav-link" href="members">
    👥 Члени сім'ї
</NavLink>
```

## 📊 Демонстраційні дані

Всі нові сторінки мають демонстраційні дані для тестування:

### Members.razor
- 10 демонстраційних членів з різними рангами

### News.razor
- 6 новин з різними категоріями

### Gallery.razor
- 8 демонстраційних фото з 3 категоріями

### Rules.razor
- Повний текст правил з 6 розділами

## 🚀 Запуск та тестування

### Запуск проекту:
```bash
cd "Raven Family"
dotnet build
dotnet run
```

### Доступ:
- **Локально:** http://localhost:5000
- **По мережі:** http://[IP]:5000

### Тестування адаптив:
- Відкрити DevTools (F12)
- Натиснути Ctrl+Shift+M
- Вибрати мобільний пристрій
